# 3D Ball Throwing Game (Unity)

This is a simple 3D ball throwing game made with Unity, inspired by bowling-style gameplay. The player throws a ball to hit and knock down objects in the scene.

## Features
- 3D physics-based ball throwing
- Bowling-like mechanics
- Built with Unity
- Simple and clean project structure

## How to Run
1. Clone or download the repository.
2. Open the project in Unity.
3. Open the main scene.
4. Press Play to run the game in the Unity Editor.

## Controls
- Mouse or touch input to aim and throw the ball (depending on implementation).

## Notes
This project was created as a practice/demo project and can be extended with scoring, levels, UI, and sound effects.
